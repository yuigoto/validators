# `@yuigoto/validators`

> Simple string validation library for personal projects.

Provides simple validation for:
- Some Brazilian documents (CPF, CNPJ, PIS);
- E-mail addresses;
- Credit card numbers;

Install it with:

    npm install --save @yuigoto/validators
